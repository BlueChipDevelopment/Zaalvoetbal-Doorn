App Installeren

Voor de beste ervaring van notificaties, installeer de app op je apparaat:

phone_iphone iPhone/iPad

  1. Tik op het deel-icoon ios_share onderaan je scherm
  2. Scroll naar beneden en tik op "Voeg toe aan beginscherm"
  3. Tik op "Voeg toe" rechtsboven

phone_android Android

  1. Tik op het menu more_vert rechtsboven
  2. Kies "App installeren" of "Toevoegen aan startscherm"
  3. Tik op "Installeren"

computer Computer

  1. Kijk naar het installatie-icoon get_app in de adresbalk
  2. Klik op "Installeren"

devices App Installeren

Zoek naar de installatie optie in je browser menu of adresbalk.

Welke notificaties ontvang je?

  • sports_soccer Notificaties voor komende wedstrijden
  • how_to_reg Verzoeken om aanwezigheid door te geven
  • group Updates over teamindelingen
